Website Features & Technologies

This page explains the key features I implemented and the technologies used to build the project. The goal is to create a clean photo-gallery experience similar to modern apps, but using simple web technologies (HTML, CSS, JavaScript).

Try Gallery Features Open Albums

Main Features

The project focuses on navigation, photo browsing, and a smooth UI. Below are the main features included.

Responsive Photo Grid Photos display in a flexible grid that adapts to mobile, tablet, and desktop screens.
Search & Filter Users can search photos by title/description and filter by album tags like Travel or Family.
Lightbox Viewer Click any photo to open it in a fullscreen viewer with title, album name, and description.
Albums Page → Filter Link Album cards open the Gallery already filtered using URL parameters (example: ?tag=travel).
Contact Form Validation Contact page checks required fields and email format, then shows a success message (demo mode).
Modern UI + VFX Background Glass UI cards, gradients, animated orbs, and subtle motion for a professional look.
