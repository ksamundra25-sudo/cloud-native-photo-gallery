About Me

My name is Samundra Khanal. I was born in Nepal, and I am currently living in the UK while studying for my BSc degree in Computing. I enjoy learning how websites and applications are built, especially how design and functionality work together. I’m interested in topics like web development, user interface design, and building projects that feel modern, simple, and easy to use.

This project helps me practice real skills like structuring pages with HTML, styling with CSS, and adding interactive features using JavaScript. My aim is to keep improving step-by-step and create work that looks professional and is easy for users to understand.

About This Website

My Photo Gallery is a mini photo-management website inspired by apps like online galleries. It includes a welcome homepage, a photo gallery page with search and a lightbox viewer, and an albums page that links to filtered views. The design uses a responsive layout so it works on mobile and desktop.

Home Welcome message, description, and quick navigation.
Gallery Grid layout, search, filters, and fullscreen image viewer.
Albums Album cards that open Gallery with a selected category.
